# AtletaData
AtletaData es una empresa tecnologica espanola especializada en la monitorizacion de
jugadores de futbol profesional.
Su producto combina dos cosas :
un sensor fisico que se coloca en la espalda del jugador dentro de su equipacion, con una
plataforma web donde el equipo tecnico puede consultar todos los datos recogidos en
tiempo real.
El sensor funciona tanto en partidos como en entrenamientos, recogiendo datos
continuamente y enviandolos a la plataforma.
